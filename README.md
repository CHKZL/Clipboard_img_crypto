# 如何使用
## 加密
* Ctrl+C复制QQ窗口里的图或者任何图片文件
* 按Alt+1加密图片
* 然后Ctrl+V粘贴到任意地方即可
## 解密
* Ctrl+C复制本程序加密的表情
* 按Alt+2解密图片
* 然后Ctrl+V粘贴到任意地方即可


## 兼容性
当前还不支持动态GIF图片，之后应该可以摸出来
开发&测试环境为 WIN10  .NET4.5
